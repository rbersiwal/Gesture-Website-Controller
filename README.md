# Gesture Website Controller

A hand gesture recognition project built using **Python**, **OpenCV**, and **MediaPipe** that allows users to control actions on their computer through hand gestures.

## Features

* 👍 Open YouTube using a hand gesture
* ✋ Open ChatGPT using a hand gesture
* ✊ Open Google using a hand gesture
* 👌 Take screenshots using an OK gesture
* Real-time hand tracking with webcam
* Gesture recognition using MediaPipe Hand Landmarker

## Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI

## Installation

Clone the repository:

```bash
git clone https://github.com/rbersiwal/Gesture-Website-Controller.git
```

Navigate to the project folder:

```bash
cd Gesture-Website-Controller
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python main.py
```

## Gesture Controls

| Gesture       | Action          |
| ------------- | --------------- |
| 👍 Thumb Up   | Open YouTube    |
| ✋ Open Palm   | Open ChatGPT    |
| ✊ Closed Fist | Open Google     |
| 👌 OK Sign    | Take Screenshot |
